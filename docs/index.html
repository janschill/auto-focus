Getting Started

1

Download & Install

Download Auto-Focus and the required "Toggle Do Not Disturb" shortcut. Install both to get started.

  • • Extract the downloaded ZIP file
  • • Drag Auto-Focus to Applications
  • • Install the shortcut from the package
2

Configure Focus Apps

Launch Auto-Focus and select which applications should trigger focus mode when you're working.

  • • Add your development tools
  • • Configure browser focus URLs
  • • Set focus thresholds and buffer times
3

Install Browser Extension

Add the Chrome extension to track focus URLs like GitHub, Linear, Figma, and other work sites.

  • • Available in auto-focus-browser-extension/
  • • Load unpacked extension in Chrome
  • • Configure focus domains
4

Start Focusing

Auto-Focus will now monitor your work and automatically manage Do Not Disturb mode.

  • • Focus for 12 minutes to trigger DND
  • • Track sessions in the insights view
  • • Access controls from the menu bar
